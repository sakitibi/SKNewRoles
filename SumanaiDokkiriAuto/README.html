SumanaiDokkiriAutoの使い方

あなたたちには
恐ろしい悪魔の病などをかける魔女になって
監視人物を気付かれ無いようにこっそり
殺したり、操ったりして下さい!

注意 ベータ版 Script APIを入れてください
ベータ版 Script APIを入れないと正しく動作しません

コマンド一覧

  • /sda:warp で監視人物にワープします 必要魔力 10
  • /sda:help でヘルプメッセージを表示します 必要魔力 0
  • /sda:fp で監視人物に毒を付与する魔法をかけます 必要魔力 20
  • /sda:testfor で監視人物がワールドに居るか確かめます 必要魔力 5
  • /sda:devilfp で監視人物に毒を付与する悪魔の魔法をかけます 必要魔力 30
  • /sda:hypnosys で監視人物を浅く眠らせる催眠術をかけます 必要魔力 30
  • /sda:devilhypnosys で監視人物を深く眠らせる悪魔の催眠術をかけます 必要魔力 50
